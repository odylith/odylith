# Support

Last updated: 2026-04-07

## Support Model

Odylith is an open-source project with best-effort maintainer support.

- There is no paid support offering.
- There is no guaranteed response-time SLA.
- Odylith is GA on supported public install platforms as of 2026-04-07.
- The latest public GitHub release is `v0.1.9`, published on 2026-04-07.

## Where To Ask For Help

### Bugs and feature requests

Use GitHub issues:

- [Bug report](https://github.com/freedom-research/odylith/issues/new?template=bug_report.md)
- [Feature request](https://github.com/freedom-research/odylith/issues/new?template=feature_request.md)

### Security issues

Do not use public issues. Follow [SECURITY.md](SECURITY.md).

### General usage and project questions

Open a GitHub issue if the question is actionable for the project and can be
answered publicly.

## What Maintainers Can Usually Help With

- reproducible install, upgrade, rollback, and repair problems
- incorrect or missing public documentation
- governance-surface generation problems
- clearly scoped feature requests tied to the public product contract

## What Is Not Guaranteed

- custom consulting for consumer-repo integration
- debugging arbitrary third-party environments with no reproduction
- guaranteed timelines for feature requests
- support for unpublished forks or heavily modified local installs
