# Odylith Brand Assets

Canonical runtime-facing Odylith brand assets.

Use:

- `icon/`
  - square mark for app chrome, cards, shell navigation, and compact UI surfaces
- `lockup/`
  - icon + wordmark for headers, dashboards, and documentation shells
- `favicon/`
  - browser-tab assets

These files were curated from the designer handoff under `docs/brand/odylith/2026-04-rebrand-package/`.

The SVG files in this directory are raster-image wrappers supplied by the design package, not editable path vectors.
