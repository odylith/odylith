# Odylith 0.1.5

Odylith now gives repos a cleaner first-run and upgrade experience.

- The starter launchpad pushes users toward one real code path, one backlog item, one registry boundary, and one atlas map.
- The upgrade spotlight reopens cleanly, points to a permanent release note, and avoids leaking stale first-install state.
- Shell refresh happens immediately after upgrade so the repo stays on the current Odylith contract.

This release also made the product language clearer at the moment users actually see it from the shell.
