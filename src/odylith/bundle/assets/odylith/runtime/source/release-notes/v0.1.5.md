# Cleaner First Run

Odylith 0.1.5 makes the first install and first upgrade feel clearer, safer, and more intentional.

- Start from one real code path, not generic setup filler.
- Reopen upgrade notes without leaking stale first-install state.
- Refresh the shell immediately after upgrade so the repo stays on the current contract.
