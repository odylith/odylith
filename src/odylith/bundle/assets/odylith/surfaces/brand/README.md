# Odylith Brand Assets

Curated copy of the approved Odylith brand asset set.

This directory is intended for product-owned surface references under
`odylith/surfaces/brand/` in repos that carry the product shell.

Source and curation flow:

- designer handoff: `docs/brand/odylith/2026-04-rebrand-package/`
- runtime-packaged canonical set: `src/odylith/runtime/surfaces/assets/brand/`

The SVG files in this directory are raster-image wrappers supplied by the approved design package.
