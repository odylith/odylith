# Odylith rebrand package

This package is the repository-canonical Odylith asset handoff used to replace the previous Odylith-predecessor brand surface assets.

## Included assets

- `icon/`
  - `odylith-icon.svg`
  - 16x16, 32x32, 48x48, 64x64, 128x128, 256x256, and 512x512 PNG exports
  - black and white 512x512 icon variants
  - `odylith-favicon.ico`
- `lockup/`
  - `odylith-lockup-horizontal.svg`
  - 1200x300 and 1600x400 horizontal exports
  - black and transparent horizontal variants
  - source preview rasters

## Important note

The approved artwork exists as raster source imagery plus image-wrapped SVG exports. A fully editable path-based vector source is not available in this package.
