# Cleaner Upgrade Handoff

Odylith v0.1.6 turns the release surface into a real product touchpoint instead of a metadata dump.

- Ship one authored note into the repo and render from that source.
- Keep the upgrade recovery affordance temporary, quiet, and out of the header.
- Tighten shipped guidance so the shell reads like a product, not internal scaffolding.
