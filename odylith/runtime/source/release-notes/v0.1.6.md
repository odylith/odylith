Odylith v0.1.6 is a release-quality cleanup aimed at the moment right after install or upgrade. Consumer repos now get the authored release-note source in `odylith/runtime/source/release-notes/`, so the spotlight and release-note page render real product copy instead of raw placeholder text or thin GitHub metadata.

This release also removes release-version clutter from the shell header, keeps the temporary reopen affordance in the bottom recovery pill for only a short post-upgrade window, and tightens shipped consumer guidance so routine Odylith grounding chatter stops leaking into end-user progress updates. On the maintainer side, the release lane now treats authored notes and stricter release proof as part of the shipping contract instead of optional cleanup.
